HTML Element Tags
Tag Name Description
<html> HTML Contains the HTML code.
<head> head Contains the metadata of the page.
<body> body Contains the main displayed elements of the page.
<title> title Contains the title of the site which is displayed in the browser tab. Usually in <head>.
<p> paragraph Used to print basic texts.
<h> heading Used to print titles, there are 6 different values (from h1 to h6). h1 is the biggest.
<div> division Used to divide the code in different section.
<span> span Used to seperate enclosed inline text from the whole.
<em> emphasize Used to render enclosed string italic.
<strong> strong Used to render enclosed string bold.
<br> line break Used to skip to ne next line
<ul> unordered list Used to create a list with bullet points.
<ol> ordered list Used to create a list with numerical order.
<li> list element Used to write the list elements.
<img/> image Used to print an image. It is a self-closing tag and ends with a forward slash (/).
<video> video Used to add a video. Like an image tag it requires a src attribute.
<a> anchor Used to add an hyperlink. You can also wrap other elements with this tag to render them as links.
<table> table See below.
<style> style Allows to write code in CSS language between the tags.
<link> link Links a CSS file to an HTML file. Includes href, type and rel attributes. Usually in <head>.

Basic HTML Attributes
Attribute Description
id Specifies an Element.
src Source of the media.
alt Alternative text of the media.
width Alters the dimension of the media.
height Alters the dimension of the media.
controls Adds basic controls to a video. The include pause, play, skip and the volume controls.
href Hyperlink reference of an hyperlink.
target Specifies how a link should open. For it to open in a new tab the value must be "_blank"
style Enables custom CSS styling to the element.
type Describes the type of document you are linking to. For CSS: "text/css"
rel Describes the relationship between the linked files. For HTML-CSS: "stylesheet"
class Assigns one or more CSS class to tan element. To assign multiple classes leave a space between them.

HTML Table Tags
Tag Name Description
<table> table Creates a table element.
<tr> table row Creates a table row.
<td> table data Creates a table cell.
<th> table heading Creates a table title cell.
<tbody> table body Contains the main table data
<thead> table head Contains table title and column headings.
<tfoot> table foot Contains the table's final rows, usually sum values.

Table HTML Attributes
Attribute Description
scope Specifies the scope (col or row) of a table heading.
colspan Specifies how many columns the element spans.
rowspan Specifies how many rows the element spans.

CSS Declarations
Declaration Description Units
border Styles the border of the element. size(px) solidity color
font-family Styles the typeface of the text. fontname
font-size Styles the size of text. size(px)
font-weight Styles the thickness of text. normal, bold or bolder
text-align Locates the position of text. left, right or center
color Styles the foreground color of an element. color
background-color Styles the background color of an element. color
opacity Styles the opacity of an element. from 0 to 1
background-image Adds an image to an elements background. url("#")
!important Can be attached to any property to increase its specifity. url("#")
